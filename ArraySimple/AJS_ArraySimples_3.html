Ciclo For y Arrays Simples

En este ejercicio el objetivo será que puedas hacer una iteración sobre el array de parrafos.

En el HTML vas a tener predefinido una estructura con un button, un div con id 'parrafos' y en JavaScript tendrás un array con parrafos.


¿Qué tiene que hacer el botón en esta actividad?

Al hacer clic en el botón se debe agregar un párrafo con cada valor de un array.


En JS:

  • Declara la función agregarParrafos(). Esta función deberá:
    • Recorrer el array parrafos para:
      • Guardar en una variable un element p que vas a crear usando el método createElement().
      • Agregar como contenido del párrafo, cada valor del array.
      • Capturar el elemento con id "parrafos" y agregarle el elemento creado, usando el método appendChild().

En HTML:

  • Agrega al botón el atributo necesario para que ejecute la función.
