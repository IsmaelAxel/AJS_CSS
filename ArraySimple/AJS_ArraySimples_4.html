Ciclo For y Arrays Simples

En este ejercicio el objetivo será que puedas hacer una iteración sobre el array de sabores.

En el HTML vas a tener predefinido una estructura con un button, una lista desordenada y en JavaScript tendrás un array con una de sabores.


¿Qué tiene que hacer el botón en esta actividad?

Al hacer clic en el botón se debe generar la lista de gustos de una heladería pero en caso de ser frambuesa, este aparecerá como "no disponible".


En JS:

  • Declara la función mostrarSabores(). Esta función deberá:
    • Recorrer el array sabores para:
      • Guardar en una variable un element li que vas a crear usando el método createElement().
      • Agregar como contenido del <li>, cada sabor del array.
      • Capturar el elemento con id "listaDeSabores" y agregarle el elemento creado, usando el método appendChild().
      • En la heladería se quedaron sin frambuesa y quieren que se muestre en rojo. Para eso, el programa, con una condición, debe comparar cada sabor para saber si es "Frambuesa".
      • Si la condición en positiva, al elemento, se le debe agregar la clase “no-disponible”. (Te recomendamos prestar atención a las mayúsculas presentes en los nombres de los sabores)

En HTML:

  • Agrega al botón el atributo necesario para que ejecute la función.
