Ciclo For y Arrays Simples

En este ejercicio el objetivo será que puedas hacer una iteración sobre el array de tu listaDeTareas.

En el HTML vas a tener predefinido una estructura con un button, una lista ordenada y en JavaScript tendrás un array con una listaDeTareas.


¿Qué tiene que hacer el botón en esta actividad?

Al hacer clic en el botón se debe agregar a una lista todos los valores de un array.


En JS:

  • Declara la función listar(). Esta función deberá:
    • Recorrer el array listaDeTareas para:
      • Guardar en una variable un elemento li que vas a crear usando el método createElement().
      • Colocar cada tarea del array como contenido del li
      • Capturar el elemento con id “tareas” y agregarle el creado, usando el método appendChild().

En HTML:

  • Agrega al botón el atributo necesario para que ejecute la función.
