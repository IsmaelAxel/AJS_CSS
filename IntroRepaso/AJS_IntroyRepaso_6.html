Introducción y repaso - 6

En este ejercicio, tu tarea será implementar la lógica indicada en la consigna para que, al hacer clic en el botón en el HTML, se ejecute una función específica.


En JS:

  • Copia el siguiente array de objetos en tu archivo script.js:

    • var jugadores = [];

      jugadores[0] = {
      nombre: "Messi",
      camiseta: 10,
      imagen: "https://i.ibb.co/vYFDYWs/gettyimages-972635212.jpg"
      }

      jugadores[1] = {
      nombre: "De Paul",
      camiseta: 7,
      imagen: "https://i.ibb.co/MhQc9DK/8-Kq-C1cmp-400x400.jpg"
      }

      jugadores[2] = {
      nombre: "Martinez",
      camiseta: 23,
      imagen: "https://i.ibb.co/G01tQz1/KSYBapv-DB-340x340-1.jpg"
      }

  • Declara la función mostrarJugador(). esta función deberá:
    • Guardar en una variable, la respuesta a un prompt que le pregunte por un número de la lista de jugadores.
    • Guardar en otra variable, el valor almacenado en la variable anterior menos 1 (-1).
    • Capturar el elemento con id “mensajeJugador” y mostrar un mensaje que incluya el uso de las propiedades nombre y camiseta del objeto. Por ejemplo: “El jugador De Paul tiene la camiseta número 7”
    • Por último, buscar el elemento con id “imagenJugador” y modificar su ruta (.src) colocando la propiedad imagen del objeto.

En HTML:

  • Agrega al botón el atributo correspondiente para que, al hacer clic, se ejecute la función que has desarrollado.
