¿Qué otras cosas podemos hacer con los arrays? Array.join()

El último método que conocerás, será .join(separador). Esté, te permite unir todos los valores del array. Entre los paréntesis, puedes escribir el símbolo que separá a cada palabra. Por ejemplo:

var verduras = ["Cebolla", "Ajo", "Apio", "Chaucha"];

var verdurasUnidas = verduras.join("-");

console.log(verdurasUnidas) //Esto mostrará "Cebolla-Ajo-Apio-Chaucha"

IMPORTANTE: Todos los métodos se aplican sobre el array, es decir colocas el nombre del arreglo, seguido, un punto y luego el método.
