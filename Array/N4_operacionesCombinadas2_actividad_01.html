Ayuda al preceptor a recordar alumnos según su número de orden

Algunas veces, el preceptor necesita conocer qué alumno es el que tiene cierto número en la lista. Ayúdalo con este programa para que pueda ingresar el número de alumno con un prompt, y pueda recibir un alert con el nombre correspondiente.

El HTML y el CSS ya están dados.


Trabaja en JavaScript

Primero debes saber quién es el alumno. Para eso, ya tienes creada la variable numeroAlumno con un prompt que pregunta por el número de alumno.

Además tienes la variable indice que guarda el verdadero número para acceder al array alumnos 

¿Por qué la variable indice guarda como valor numeroAlumno - 1 ?
            Recuerda que los índices de los arrays empiezan en 0, pero los alumnos en la lista están numerados desde 1.

Ahora debes componer el alert para mostrarle el nombre del alumno. El mensaje debe ser como el siguiente: Juan tiene el número de orden 8

Entonces el alert tendrá este formato: variable + " texto " + variable

Debes reemplazar "Juan" y "8" por los datos correspondientes.

Para obtener el nombre del alumno, debes acceder al array alumnos con el valor de índice adecuado, y también deberás utilizar la propiedad correspondiente al nombre.

Y ya tienes el número de alumno almacenado en una variable. Lo ingresó el preceptor en el prompt.
