¿Sabías qué...? Los objetos pueden realizar acciones.

Hasta ahora conoces la sintaxis típica de un objeto. Mira el siguiente ejemplo de un objeto gato:

var gato = {
    nombre: "Michifus",
    color: "naranja",
}

Lo que no sabías es que a este objeto gato puede maullar. Te preguntarás, ¿cómo? ¡Con funciones! Mira el ejemplo:

var gato = {
    nombre: "Michifus",
    color: "naranja",
    maullar: function () {
        alert("Miau miau")
    }
}

En el objeto gato, nombre y color son propiedades. Si recuerdas, para poder mostrar estas propiedades solo hacía falta llamar al objeto, seguido un punto y luego la propiedad. Así sería si lo muestras en la consola:

console.log(gato.nombre)//Esto muestra "Michifus"

Ahora, maullar es una función, entonces para ejecutarla vas a llamar al objeto y luego a la función. Así:

gato.maullar()//Esto mostraría un alert que diga "Miau miau"

Ahora te toca a vos.

Crea el objeto perro con las siguientes propiedades:

nombre: "Firulais",

color: "negro"

Además deberás agregar la función ladrar. Esta función ejecutará un alert que diga "Guau guau"

Por último, ejecuta la función ladrar del objeto perro.
