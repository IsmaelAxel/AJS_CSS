Ingreso por Contraseña

En este ejercicio usarás un condicional if/else para validar el ingreso a un sitio según una contraseña preestablecida.


Programa en JS

  • Declara una variable llamada passwordCorrecto y guarda dentro de ella el texto "zxc#9192"
  • Crea una variable llamada respuestaUsuario y guarda dentro de ella la respuesta a un prompt() con el siguiente mensaje: "¡Hola! Ingresa la clave de acceso para continuar:"
  • Crea un condicional que compare si el valor que quedó guardado en la respuestaUsuario es igual al valor de la variable passwordCorrecto
  • Si la condición se cumple, muestra un alert() con el siguiente mensaje: "¡Correcto! Puedes continuar"
  • Si la condición no se cumple, en el bloque else{} debes mostrar un alert() con el siguiente mensaje: "Acceso denegado"
