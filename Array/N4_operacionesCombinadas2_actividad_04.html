Ya tienes dado el array alumnos que tiene los promedios ya calculados. Obsérvalo para familizarizarte con su estructura. ¿Qué propiedades puedes observar en cada alumno?

Primero debes saber quién es el alumno. Para eso, ya tienes la variable numeroAlumno con un prompt que pregunta por el numero de alumno a modificar.

Debajo crea la variable indice, que contenga el índice del estudiante seleccionado según el dato ingresado por el usuario en el prompt. Ten en cuenta que el usuario ve los números de alumnos numerados desde 1, pero el array numera sus índices desde 0.
