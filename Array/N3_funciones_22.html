Área de un triángulo

En este ejercicio deberás crear una función para realizar el cálculo del área de un triángulo con datos ingresados por el usuario.


Prepara el HTML:

  • El botón "¡Calcular!" debe ejecutar la función obtenerAreaTriangulo()

Programa en JS:

  • Declara una función llamada obtenerAreaTriangulo() - dentro de la función:
    • Pregunta al usuario la medida de la base del triángulo. No olvides guardar el dato para poder utilizarlo luego.
    • Pregunta al usuario la medida de la altura del triángulo. No olvides guardar el dato para poder utilizarlo luego.
    • Realiza el cálculo necesario para obtener el área de un triángulo: base*altura/2
    • Muestra en un alert() únicamente el valor del área del triángulo
