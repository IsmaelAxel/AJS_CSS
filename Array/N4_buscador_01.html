Buscador de objetos - Paso 1

Te damos la bienvenida a la actividad integradora de Estructura de Datos. Aquí trabajarás tanto conceptos de HTML y CSS como también los que aprendiste recientemente, como ser objetos, arrays, la combinación de ambas estructuras y el método find().

El objetivo es que crees una base de datos para que el sitio pueda mostrar imágenes y textos, dependiendo de lo que el usuario busque. En otras palabras, armarás un buscador y su lógica para que muestre el resultado.
