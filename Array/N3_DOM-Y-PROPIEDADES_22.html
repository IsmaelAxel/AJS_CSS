Votación

En este ejercicio crearás un menú para registrar la opinión de un usuario


Prepara el HTML

  • Coloca un <h1> con el texto Votación
  • Coloca un <p> con el texto ¿Estás de acuerdo con la propuesta?
  • Coloca un menú <select> con id="menuVotacion" - dentro del menú:
    • Una opción con value "No Contesta" el texto --- (tres guiones) y el atributo disabled selected - mira el ejemplo:
        • <option value="No Contesta" disabled selected> --- </option>
      • Una opción con value "SÍ" y el texto SÍ
      • Una opción con value "NO" y el texto NO
      • Coloca un <button> asociado a la función registrarVoto() con el texto Votar

Programa en JS

  • Declara la función registrarVoto() - dentro de la función:
    • Crea una variable llamada votoUsuario y guarda dentro de ella el valor de la opción seleccionada por el usuario en el menú con ID "menuVotacion"
    • Muestra en un alert() el siguiente mensaje concatenado: "Voto registrado: " + votoUsuario
