Cambiando Imágenes

En este ejercicio deberás crear el código JS necesario para cambiar imágenes que ya se encuentran en un sitio.

Sigue los pasos a continuación:


Prepara el HTML:

  • Agrega el atributo "onclick" al botón con id "switcher" para que al hacer click se ejecute la función "cambiarImagen()"

Programa en JS:

  • Declara la función cambiarImagen() - Dentro de ella, sigue los siguientes procedimientos:
  • Declara la variable nuevaUrl que guarde un prompt() para pedirle al usuario la URL de la nueva imagen. Puedes elegir el mensaje que quieras para el prompt()
  • Selecciona la <img> con id "imagen" y cambia su ruta actual, por la nueva ruta, que está guardada en la variable llamada nuevaUrl
