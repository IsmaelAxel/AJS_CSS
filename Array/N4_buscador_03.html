Buscador de objetos - Paso 3

En esta parte, vas a crear el sustento de este buscador. LA BASE DE DATOS. Estará compuesta por un array de objetos con sus propiedades.

Recuerda que la temática del buscador puede ser la que quieras, siempre que cumplas con lo pedido en las consignas.

¡Manos a la obra!


En JS:

  • Declara la variable miCatalogo que contenga un array vacío.
  • Dentro de este array, debe haber como mínimo 4 objetos. En cada posición del array miCatalogo, crea un objeto con las siguientes propiedades completas: nombre, imagen y descripcion. Todas son propiedades de tipo string, es decir deben ir entre comillas. Recuerda que como mínimo deben ser 4 objetos pero, puedes colocar más si así lo quieres.

Aquí te dejamos un modelo de como debería quedar completo el objeto:

miCatalogo[0] = {
      nombre: "AuronPlay",
      imagen: "https://i.ibb.co/nwzS2Yv/16315244430699.jpg",
      descripcion: "Su saludo de cabecera es: ¡Eeeeey! Pero que pasa chavales.
¿Todo bien? ¿Todo correcto? Y yo que me alegro."
};
