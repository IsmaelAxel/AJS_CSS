¿Qué otras cosas podemos hacer con los arrays? Array.shift()

El segundo método que verás será .shift(). Esté, te permite eliminar el primer valor del array. Por ejemplo:

var verduras = ["Cebolla", "Ajo", "Apio", "Chaucha"];

verduras.shift();

console.log(verduras) //Esto mostrará ["Ajo", "Apio", "Chaucha"]

IMPORTANTE: Todos los métodos se aplican sobre el array, es decir colocas el nombre del arreglo, seguido, un punto y luego el método.
