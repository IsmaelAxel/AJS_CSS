Creando un array [II]

En el ejercicio anterior, creaste una variable que contenía una arreglo con los nombres de tus mejores amigos

Ahora, vas a crear otro array pero que en esta ocasión guarde números.


En JS:

  • Para eso, crea una variable que se llame cumpleaniosAmigos (recuerda que no puedes usar caracteres especiales) y dentro de ella, guarda un array que contenga los días de los cumpleaños de tus mejores amigos.
