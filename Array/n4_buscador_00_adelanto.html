Buscador de objetos - Adelantando el Array

Te proponemos ir empezando a transformar la información que recolectaste en tu Google Sheet a un array de objetos.


En JS:

  • Declara la variable miCatalogo que contenga un array vacío.
  • Dentro de este array, debes colocar objetos. En cada posición del array miCatalogo, crea un objeto con las siguientes propiedades completas: nombre, imagen y descripcion. Todas son propiedades de tipo string, es decir deben ir entre comillas.

Aquí te dejamos un modelo de como debería quedar completo el objeto:

miCatalogo[0] = {
      nombre: "AuronPlay",
      imagen: "https://i.ibb.co/nwzS2Yv/16315244430699.jpg",
      descripcion: "Su saludo de cabecera es: ¡Eeeeey! Pero que pasa chavales.
¿Todo bien? ¿Todo correcto? Y yo que me alegro."
};
