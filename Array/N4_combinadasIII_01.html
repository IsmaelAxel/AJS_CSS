¿Cuál es la capital de...?

En esta actividad, practicarás el uso de la función find(). Tu tarea será desarrollar un programa que permita al usuario buscar la capital de varios países de Latinoamérica.

Encontrás en el archivo script.js un array llamado listadoPaises. Este contiene objetos de tipo país, cada uno con las propiedades nombrePais y capital.


En JS comienza por:

  • Crear la variable paisBuscado que guarde la respuesta de un prompt que pregunte "Ingrese un país de Latinoamérica para conocer su capital."
  • Luego, define la variable objetoEncontrado. En esta variable, debemos guardar el resultado de la búsqueda. Para eso aplica el método find() sobre el array listadoPaises para que busque la coincidencia entre el paisBuscado y la propiedad nombrePais del objeto en cuestión. Debería quedar algo así:
var objetoEncontrado = listaPaises.find(objeto => objeto.nombrePais == paisBuscado);
  • Por último, deberás mostrar un alert mostrando una oración aclarando el país que se buscó y su capital. El mensaje debería quedar algo así: "La capital de " + objetoEncontrado.nombrePais + " es " + objetoEncontrado.capital
