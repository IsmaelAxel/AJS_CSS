Validando "Captchas"

En este ejercicio crearás una función para validar que el usuario complete correctamente un campo <input> según una imagen "captcha" mostrada.


Prepara el HTML

  • Al botón Verificar, colócale el atributo necesario para que se ejecute la función "validarCaptcha()"

Programa en JS

  • Declara la función llamada validarCaptcha() - Dentro de la función:
    • Crea una variable llamada intentoUsuario y guarda dentro de ella el value de lo que haya ingresado el usuario en el <input> con id "inputCaptcha"
    • Crea un condicional que compare si el valor que quedó guardado en intentoUsuario es igual al texto deseado: "QwErTy123"
    • Si la condición se cumple, muestra un alert() con el mensaje "¡Bien! Eres humano"
    • Si la condición no se cumple, en el bloque else{}, muestra un alert con el mensaje "Intenta de nuevo por favor"
