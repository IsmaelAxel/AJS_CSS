Ciclo For y Arrays de Objetos

En esta clase el objetivo es que puedas definir programas que muestren información de las estructuras de datos.

La estructura HTML básica ya está creada y los arrays de objetos ya están cargados, es importante que no los borres. Si lo haces tus programas no funcionarán.

¿Qué tiene que hacer el programa en esta actividad?

El programa debe mostrar textos e imágenes en HTML provenientes de objetos dentro de un array.


En JS:

  • Declara la función mostrarAmigos(). Esta función deberá:
    • Recorrer el array amigos y:
      • Crear una imagen, un div y un h3 usando el método createElement(), y guardarlos en distintas variables para poder utilizarlos después.
      • Colocar cada nombre como contenido del h3 y cada imagen como ruta de la img.
      • Agregar el elemento img y el h3, dentro del div creado anteriormente.
      • Capturar el elemento con id "actividad02" y agregarle el elemento div, usando el método appendChild().

En CSS:

  • Dale al elemento con id actividad02 las propiedades necesarias para que el diseño se vea en 2 columnas y 3 filas.

En HTML:

  • Agrega al botón el atributo necesario para que ejecute la función.
